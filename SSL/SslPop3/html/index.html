SSL POP3 Email Example
This example program demonstrates how to get email from a SSL POP3 server.
To run the example:
  • Verify you have valid settings for the IP Gateway and DNS fields shown below
  • The default SSL port number for POP3 mail servers is 995
  • To enable DHCP, set the IP address field is 0.0.0.0 using IPSetup
  • Once all the data has been entered, click on the "Get Mail" button
