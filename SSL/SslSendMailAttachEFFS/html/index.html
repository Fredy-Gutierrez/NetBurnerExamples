Send Mail With EFFS Attachment Example Application

This example program demonstrates how to send a EFFS FAT32 email attachment from a NetBurner device. To run the example:
  • Verify you have valid settings for the IP Gateway and DNS fields shown below.
  • To enable DHCP, set the IP address field is 0.0.0.0 using IPSetup.
  • Enter data into the form. SMTP authentication will be used for all email messages, and a password must be provided.
  • If you enter text (256 chars max) in the Create Text Attachment field a text file
    attachment, TextFile.txt, will be created.
  • If you specify a file name in the SD file name field, the file must exist on
    the SD card.
  • This application supports FTP. We recommend that you use an FTP client to view and
    modify the files on the SD card.
  • Once all the data has been entered, click on the "send mail" button.
  • When trying to connect to a Google Account, you will likely need to enable less secure
    application access. If you use two-factor authentication with your account, you will
    need to setup an application password for the account being connected to. This can be done at
    the following URL: https://myaccount.google.com/apppasswords
