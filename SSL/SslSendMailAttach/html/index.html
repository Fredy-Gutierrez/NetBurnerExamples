Send Mail Example with non-EFFS Attachments

This example program demonstrates how to send email attachments from a NetBurner device without using the EFFS FAT32 file system.
To run the example:
  • Verify you have valid settings for the IP Gateway and DNS fields shown below
  • To enable DHCP, set the IP address field is 0.0.0.0 using IPSetup
  • Enter data into the form
  • This example only works with Authenticated mail servers, so you must enter a password
  • Text data in the "Text for Attachment" field will be attached to the email message at a .txt file
  • Once all the data has been entered, click on the "send mail" button
