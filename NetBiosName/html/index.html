NetBios Name Example

This example program demonstrates how you can use the NetBios service to address
your NetBurner device by name as well as IP address. Note that NetBios is different
than DNS, and will only work on your local LAN, not the Internet.

The NetBios Name can contain any alphanumeric character except:
spaces, \ : / * ? ; | . It must be 15 characters or less.
