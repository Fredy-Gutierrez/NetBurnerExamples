/** @pagegroup{Configuration, WebConfig, NetBurner Config Web Interfaces}
The examples in this directory show how to create utilize a web interface with NetBurner's config system.

   - BasicWebConfig - Shows a minimal web interface that will display and allow the modification of config
     data that is recieved in the form of a JSON blob. This example is barebones HTML, CSS, and JavaScript;
     No external libraries are required.
   - CustomWebConfig - Shows a rich web interface that will display and allow for the modification of config
     data that is recieved in the form of a JSON blub. However, this example uses jQuery and BootStrap to
     provide additional styling.
*/
