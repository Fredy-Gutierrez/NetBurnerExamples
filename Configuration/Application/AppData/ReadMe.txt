/** @pagegroup{Configuration, AppData, NetBurner Config and Application Data}
The examples in this directory show how to create, modify, and store persistant data values through the
configuration system that can be used in user applications.

   - BasicConfigVariables - Shows how to create, modify and store basic config variable types with the
     config system
   - ConfigClass - Expands on BasicConfigVariables, and shows how to create new class types that inherit
     from base config objects, and how they can be combined together to create complex objects
   - NestingConfigObjects - Expands on ConfigClass, and shows how config objects can be grouped together
     to create a nested hierarchy of data
